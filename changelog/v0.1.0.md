# v0.1.0

Initial extraction from illo3d.

- Disk-backed Drive v3 + Sheets v4 emulator:
  mirrored folder tree, one CSV per sheet tab,
  human-readable `_index.json`, deterministic ids.
- Fail-loud boundary outside the modelled surface.
- Fetch-level handler, HTTP server with CORS, Docker image.
- Spec-driven repository: `specs/features/`, `specs/decisions/`,
  semantic line breaks, British English.
- 0BSD licence.
